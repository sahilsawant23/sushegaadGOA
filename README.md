# 🌴 Sushegaad GOA – AI-Powered Tourism & Travel Platform

Sushegaad GOA is a full-stack tourism platform that helps travelers discover the best of Goa through AI-powered trip planning, destination exploration, real-time events, and seamless tour booking.

## ✨ Features

- 🤖 AI-powered itinerary generation using Google Gemini
- 🏖️ Explore beaches, forts, waterfalls, temples, churches, and hidden gems
- 📍 Interactive destination pages with maps and travel information
- 📅 Real-time event discovery
- 🏨 Tour and package booking system
- 👤 User authentication and profile management
- 🛠️ Admin dashboard for managing destinations, bookings, and users
- 📱 Responsive design for desktop and mobile devices

## 🛠️ Tech Stack

**Frontend**
- React
- Vite
- TypeScript
- Tailwind CSS
- Framer Motion

**Backend**
- Node.js
- Express.js
- REST APIs

**Database**
- MySQL

**AI & External APIs**
- Google Gemini API
- OpenStreetMap / Overpass API
- Ticketmaster API

## 🎯 Project Goal

To build an intelligent tourism platform that delivers personalized travel experiences while promoting Goa's attractions through AI-driven recommendations and an intuitive booking system.
